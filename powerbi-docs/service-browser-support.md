# <a name="supported-browsers-for-power-bi"></a>Power BI 支持的浏览器
在以下浏览器中运行 Power BI：

* Microsoft Edge
* Internet Explorer 11
* Chrome 桌面最新版本
* Safari Mac 最新版本
* Firefox 桌面最新版本

可用的所有平台都支持这些浏览器。
