# <a name="duplicate-a-report-page-in-power-bi"></a>在 Power BI 中复制报表页面
1. 在 Power BI 中，如果你还没有报表，可以在“[编辑视图](service-reading-view-and-editing-view.md)”中打开一个报表并[创建可视化效果](power-bi-report-add-visualizations-i.md)。 
2. 从顶部控件条中选择**复制此页**图标。
   
   ![](media/power-bi-report-copy-paste-page/pbi_duplicate_new.png)
   
   你的新页面已创建并成为活动页面。
3. 可选择[重命名报表页](service-rename.md)。
