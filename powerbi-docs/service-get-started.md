# <a name="get-started-with-power-bi-service"></a>Power BI 服务入门
本教程将帮助你开启使用 ***Power BI 服务*** 之旅。 若要了解 Power BI 服务如何适应其他 Power BI 产品/服务，我们强烈建议你先阅读[什么是 Power BI](guided-learning/gettingstarted.yml#step-1)。

![](media/service-get-started/power-bi-components.png)

Power BI 服务提供免费版本和 Pro 版本。 无论你正在使用哪个版本，都可打开浏览器并键入 www.powerbi.com 开始使用。 如果已注册，则选择在右上角看到的“**登录**”链接。 如果尚未注册 Power BI 服务，则改为选择“**免费注册**”链接。

![](media/service-get-started/power-bi-sign-up.png)

如果你需要有关使用 Power BI Desktop 的帮助，请参阅 [Desktop 入门](desktop-getting-started.md)。 如果正在寻找有关 Power BI 移动端的帮助，请参阅[适用于移动设备的 Power BI 应用](mobile-apps-for-mobile-devices.md)。

Microsoft Power BI 可帮助你及时掌握对你重要的信息。  借助 Power BI 服务，***仪表板***可帮助你对企业状况了如指掌。  你的仪表板显示各种***磁贴***，你可单击这些磁贴打开***报表***来进一步了解详情。  连接到多个***数据集***将所有相关数据组合在一起。 是否需要了解构成 Power BI 的构建块的帮助？  请参阅 [Power BI - 基本概念](service-basic-concepts.md)。

如果你在 Excel 或 CSV 文件中具有重要数据，你可以创建 Power BI 仪表板以便随时随地掌握最新信息，并与他人分享自己的见解。  你是否订阅了 SaaS 应用程序（如 Salesforce）？  从[连接到 Salesforce](service-connect-to-salesforce.md) 开始，通过该数据自动创建仪表板或[查看你可以连接到的所有其他 SaaS 应用程序](service-get-data.md)。 如果你是组织成员，请查看是否已向你发布任何[应用](service-create-distribute-apps.md)。

了解所有其他[获取 Power BI 数据](service-get-data.md)的方式。

## <a name="step-1-get-data"></a>步骤 1：获取数据
下面举例说明如何从 CSV 文件中获取数据。 想要学习此教程吗？ [下载此示例 CSV 文件](http://go.microsoft.com/fwlink/?LinkID=521962)。

1. [登录 Power BI](http://www.powerbi.com/)。 还没有帐户？ 别担心，可以免费注册。
2. Power BI 将在浏览器中打开。 在左侧导航窗格底部选择“获取数据”。
   
   ![](media/service-get-started/getdata3.png)
3. 然后，选择“文件”。 
   
   ![](media/service-get-started/gs1.png)
4. 选择“本地文件”，浏览到计算机上的该文件，然后选择“打开”。
   
   ![](media/service-get-started/gs2.png)
5. 在本教程中，我们将选择“导入”，以将 Excel 文件添加为数据集，然后就可以使用它来创建报表和仪表板。  
   
   > [!NOTE]
   > 如果选择“上传”，则整个 Excel 工作簿将上传至 Power BI，然后可以在 Excel Online 中打开它并进行编辑。
   > 
   > 
   
   ![](media/service-get-started/power-bi-import.png)
6. 数据集准备就绪后，选择“查看数据集”在报表编辑器中打开它。 ![](media/service-get-started/power-bi-gs.png).
   
   > [!TIP]
   > 进行[简要了解](service-the-report-editor-take-a-tour.md)是熟悉报表编辑器的一个不错的方法
   > 
   > 

## <a name="step-2-start-exploring-your-dataset"></a>步骤 2 ：着手了解你的数据集
现在，你已连接到数据，请尽情了解来发现各种见解吧。  找到你想要监视的内容后，你可以创建仪表板跟踪最新更改。

1. 选择仪表板上的数据集映像来浏览你刚刚连接到的数据，或者在“**数据集**”标题下，选择数据集名称打开它。 这将以空白报表的方式打开数据集。
   
   ![](media/service-get-started/power-bi-report-editor.png)
   
   > [!NOTE]
> 了解数据的另一种方法是使用“快速见解”。  有关详细信息，请参阅[快速了解简介](service-insights.md)
   > 
   > 
2. 在页面右侧的“字段”列表中，选择要构建可视化对象的字段。  选中“**销售总额**”和“**日期**”旁边的复选框。
   
   ![](media/service-get-started/fields.png)
3. Power BI 会分析数据并创建可视化对象。  如果先选择“日期”，你将看到一个表格。  如果先选择“销售总额”，你将看到一个图表。 切换到不同的数据显示方式。 请尝试通过选择折线图选项更改为折线图。
   
   ![](media/service-get-started/gettingstart5new.png)
4. 如果仪表板上有你需要的可视化效果，将鼠标悬停在相应的可视化效果之上，然后选择“**固定**”图标。  当你固定此可视化对象时，它将存储在你的仪表板上，由此你便可以大致跟踪最新值。
   
   ![](media/service-get-started/pinnew.png)
5. 由于这是新报表，因此你需要先保存该报表，然后才能将可视化对象从该报表固定到仪表板。 为报表提供一个名称（例如“按时间的销售额”），然后选择“保存并继续”。 
   
   ![](media/service-get-started/pbi_getstartsaveb4pinnew.png)
   
   新报表将显示在导航窗格的“报表”标题下。
6. 将磁贴固定到现有仪表板或新仪表板。 
   
   ![](media/service-get-started/power-bi-pin.png)
   
   * **现有仪表板**：从下拉列表中选择仪表板的名称。
   * **新仪表板**：键入新仪表板的名称。
7. 选择“固定”。
   
   会显示一条成功消息（右上角附近），告知你可视化效果已作为磁贴添加到你的仪表板中。
   
   ![](media/service-get-started/power-bi-pin-success.png)
8. 选择“转到仪表板”，通过固定的磁贴查看新仪表板。 将折线图作为磁贴固定到仪表板。 通过[重命名、调整大小、链接和重新定位磁贴](service-dashboard-edit-tile.md)的方式优化仪表板。
   
   ![](media/service-get-started/power-bi-new-dashboard.png)
   
   在仪表板上选择新的磁贴，以便随时返回到报表。

## <a name="step-3-continue-exploring-with-qa-natural-language-querying"></a>步骤 3：继续研究问题解答（自然语言查询）
1. 要快速浏览数据，请尝试在问题解答框中进行询问。 问题解答的问题框位于仪表板的顶部。 例如，尝试键入“哪个市场的收入最高”。
   
   ![](media/service-get-started/powerbi-qna.png)
2. 选择固定图标 ![](media/service-get-started/pbi_pinicon.png) 还可在你的仪表板上显示此可视化对象。
3. 将可视化对象固定到“财务示例”仪表板。
   
    ![](media/service-get-started/power-bi-pin2.png)
4. 选择“退出问答” ![](media/service-get-started/pbi_qabackarrow.png) 的后退箭头返回到仪表板，可以看到新的磁贴。
